# Register a contact - spec-driven development demo

This is an example project to demo the use of spec-driven development for creating a new web app.

Execute these prompts in the following order

`brainstorm.prompt.md`

`create-implementation-plan.prompt.md`

`implement.prompt.md`

Also see the custom instuctions at `.github/instructions/projectname.instructions.md`.
